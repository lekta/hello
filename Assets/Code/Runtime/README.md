Runtime
========

Исполняемый на клиенте модуль, основная игровая логика

### <p><b><center> Особенности </center></b></p>
- Тут используются монобехи, вызываются апдейты и реакции на ивенты
- Вьюхи и UI это тоже тут
- Подпапки это фичи (либо группа фич)
- Если фича небольшая, она может быть в подпапке целиком
- Если есть сложная логика, её лучше вынести в доменный модуль
- Какая-то уникальная логика, требующая изоляции, может быть в подпапке с асмдефом
