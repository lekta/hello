Base
========

Модуль с базовым кодом

### <p><b><center> Особенности </center></b></p>
- Самый нижний слой проекта, не зависит от прочих модулей
- Может зависеть от классов юнити и пакетов

### <p><b><center> Содержит </center></b></p>
- Утилитарные функции: математика, алгоритмы
- Логику фич в отрыве от вьюх
- Расширения для системных/юнити типов
